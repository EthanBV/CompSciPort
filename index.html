What is HTML and why is it Important

HTML is a computer coding language used to create websites everywhere (pretty much all of them). Some HTML features include:

  • The abylity to easily import Javascript and CSS (We'll get to these later.)
  • Relative Simpleness and the abuylity to start learning fairly easily.
  • Many Pre-created Elemnets and Components that make web-siting easy!

What is CSS and why is it Important

CSS is a computer programming language used to add style to your HTML website, it allows things such as:

  • Coloring font,backgrounds,borders and more!
  • Asthetic elements such as pretty borders, margins, spacing, sizing etc.
  • The abylity to turn a messy skeleton of a website into a masterpeace!

Tags! Tags1 Tags!

Tags, aka elements,are any itme in your HTML code that ends and begins with a < and >. Almost all tags must be closed with a coresponding closing tag. Some tags include:

  • The < 'div' > tag is used to declare a division, containing text, other tags or pretty much anything.
  • The < 'p' > Tag is used to create a paragraph, containg text.
  • The < 'h1' > tag is used to make a header, such as a title, 'Tags! Tags1 Tags!' above is created using a headder tag.
  • The < 'img' > tag is used to add a picture to a webpage, simly fill the image URL into the src atrribute (will be explained shortly.)

Attributes

Attributes are CSS elemts that make HTML look good, they can be used to change the color, size, style, or family of font

Attributes can be added two ways, eather within the HTML by adding: style='attribute:value' or in the CSS by adding attribute:value;.
